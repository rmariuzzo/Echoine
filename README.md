Echoine Web App
===============

*Echoine* is a web app that echoes all HTTP requests to response body and log files (via [log4net](https://github.com/apache/log4netâ€Ž)), useful for testing purposes.

Features
--------

 * Echoes all incoming requests (that's the reason to be).
 * Write all incoming requests and their responses to log file.
 * Track all form data sent and write them to log file.
